# 版本记录

## 4.2.44 (4062) - 2026-05-14

- 新增网络配置页导入功能：支持从文件导入 QtEasyTier (PC端) 导出的 JSON 配置文件，兼容 PC端与 Android 端的配置互通。
- 网络配置页补齐 PC 端协议类型完整支持：新增"默认协议"选项（自动/udp/tcp/wg/ws/wss）与"加密算法"选项（aes-gcm/xor/chacha20/aes-gcm-256/openssl-aes128-gcm/openssl-aes256-gcm/openssl-chacha20），导入后可视化编辑与保存。
- 强化导入兼容性：对 `default_protocol` 与 `encryption_algorithm` 做白名单规范化，确保来自 PC 端 JSON 的协议字段在 Android 端稳定落库与运行，不受非法值影响。
- Android 网络页新增“专家模式”且全部接入真实配置生效链路：DHCP、无TUN、P2P/打洞、IPv6、加密、DNS、系统转发、UDP广播中继、监听地址/子网代理CIDR/自定义路由/白名单/TUN设备名/MTU 等项会即时保存并直接参与启动。
- 网络密钥默认隐藏，并增加小眼睛图标切换明文/密文。
- 服务器页隐藏公共节点 `tcp://183.230.36.171:11010`，避免继续显示“重庆移动跨网”节点。
- 深色模式下顶部导航栏标题统一使用内容色，修复左上角文字未反转的问题。
- 修复深色模式下切换主题后顶部状态栏图标颜色未实时更新的问题：将 `SideEffect` 替换为 `LaunchedEffect` 监听主题变化。
- 服务器页面屏蔽 ID 为 183 的公共节点不在列表中显示。
- 设置页面"服务基于 EasyTier"条目新增 easytier.cn 网站入口。

## 4.2.34 (4052) - 2026-05-13

## 4.2.26 (4046) - 2026-05-13

## 4.2.24 (4044) - 2026-05-13
